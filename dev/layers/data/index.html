Data

The data field of a layer contains the dataset that will be used to populate the plot. There are no type restrictions on this dataset, as long as it respects the Tables interface. In particular, any one of these formats should work out of the box.

The data helper function creates an under-defined layer, where only the data field is populated..

using SplitApplyPlot
df = (a = rand(10), b = rand(10))
data(df)
SplitApplyPlot.Layer((), (a = [0.3709742338435784, 0.8350384485447162, 0.09083515268981057, 0.22424521177791945, 0.5893738903773209, 0.8782699799864995, 0.6378076146333378, 0.22785573533613745, 0.3098072960830007, 0.510659984075361], b = [0.2139053598211369, 0.16909013918445814, 0.7912204677979631, 0.4129609213646994, 0.8770757297724152, 0.0037438593201715165, 0.11785408665118546, 0.26671819289805687, 0.505286268232775, 0.30238112404862605]), SplitApplyPlot.Arguments(Any[], Dict{Symbol, Any}()))
